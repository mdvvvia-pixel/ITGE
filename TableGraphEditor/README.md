# Table-Graph Editor for MATLAB

**Версия:** 1.0.0  
**MATLAB:** R2021a+  
**Фреймворк:** App Designer

---

## Описание

Интерактивное приложение для редактирования числовых таблиц через графики в стиле Excel 2003. Позволяет загружать матрицы из workspace MATLAB, отображать их в таблице и на графике, редактировать данные как в таблице, так и путем перетаскивания точек на графике.

### Запуск приложения

## Основные функции

- ✅ Загрузка числовых матриц из workspace
- ✅ Отображение данных в таблице и на графике
- ✅ Редактирование данных в таблице
- ✅ Перетаскивание точек на графике
- ✅ Два режима построения графиков (по столбцам/строкам)
- ✅ Выбор части данных для построения графика
- ✅ Автоматическая синхронизация таблицы и графика
- ✅ Сохранение изменений в workspace

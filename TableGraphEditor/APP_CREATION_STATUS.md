## Ответ на вопрос: Можно ли создать .mlapp программно?

### Краткий ответ: **Частично**

Я **не могу напрямую** создать файл `.mlapp` через App Designer GUI, так как:
1. App Designer - это графический интерфейс, требующий интерактивного взаимодействия
2. Файлы `.mlapp` - это бинарные ZIP архивы со сложной внутренней структурой
3. Нет публичного API для программного создания .mlapp файлов

## Что требует ручного выполнения

### Создание файла TableGraphEditor.mlapp

**Необходимо выполнить вручную:**

1. Откройте MATLAB
2. Выполните: `appdesigner`
3. File → New → App (выберите Blank App)
4. Сохраните как: `TableGraphEditor.mlapp` в папку `src/`
5. Следуйте инструкциям в `SETUP_INSTRUCTIONS.md`

**Или используйте скрипт для помощи:**
```matlab
cd('TableGraphEditor/src')
create_app_structure
% Скрипт предложит открыть App Designer, если файл уже существует
```

## Альтернативные подходы

### Подход 1: Использование созданных шаблонов

Я создал `TableGraphEditorBase.m`, который содержит всю структуру Properties. Вы можете:
1. Создать .mlapp файл в App Designer
2. Скопировать Properties из `TableGraphEditorBase.m`
3. Использовать `APP_STRUCTURE_TEMPLATE.md` как справочник

### Подход 2: Программное создание UI (без App Designer)

Можно создать приложение программно используя `uifigure`, но это:
- Потребует больше кода
- Не будет использовать преимущества App Designer
- Не соответствует требованиям проекта (требуется App Designer)

### Подход 3: Использование MATLAB API (экспериментально)

Я создал `auto_create_app.m`, который пытается использовать внутренние API MATLAB, но:
- Эти API могут быть недоступны или недокументированы
- Структура .mlapp файла сложна и может изменяться
- Рекомендуется использовать GUI App Designer
